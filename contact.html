 BioMart   Contact


Please choose the approapriate mailing list when contacting BioMart team.
Due to ongoing problems with spam mart-dev is now SUBSCRIBERS ONLY.
  • mart-announce - BioMart announcements
  • mart-dev - BioMart users, developers, code and installation [Archive]
If you wish to subscribe to BioMart mailing lists please send an email with an empty subject line to majordomo@ebi.ac.uk with one of the following in the message body:
  • subscribe mart-announce
  • subscribe mart-dev
To unsubscribe please send an email with an empty subject line to majordomo@ebi.ac.uk with one of the following in the message body:
  • unsubscribe mart-announce
  • unsubscribe mart-dev

If you would like to advertise your BioMart website, please drop us an email. We will be happy to hyperlink it from our home page and provide access through our Central Portal.

NB. BioMart is a software development project and we are very happy to provide as much support as we can on various aspects of BioMart installation, troubleshooting etc.
For all the data related and other end user questions concerning individual BioMart instances you should contact the appropriate team directly e.g. Ensembl, HapMap, WormBase etc e.g. helpdesk@ensembl.org.

For any other enquiry please contact Arek Kasprzyk arek.kasprzyk@oicr.on.ca
